# 教学设计智能体

一个基于 DeepSeek AI 的教学设计助手，能够为不同的教学方法（如翻转课堂、项目式学习等）自动生成教学过程和 PPT 演示文稿大纲。

## ✨ 功能特点

- 🎯 支持多种教学方法（翻转课堂、项目式学习、探究式学习等）
- 📋 自动生成详细的教学设计（教学目标、重难点、教学环节）
- 📊 自动生成配套的 PPT 演示文稿大纲
- 💬 流式输出，实时查看生成过程
- 💾 支持下载生成的内容为 Markdown 文件
- 🎨 美观的用户界面

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
streamlit run teaching_agent.py
```

### 3. 使用步骤

1. 在侧边栏输入你的 **DeepSeek API Key**
2. 填写教学信息：
   - 学科/课程名称
   - 授课内容
   - 教学对象
3. 选择教学方法和课时时长
4. 点击"生成教学设计"按钮
5. 查看生成的教学设计后，点击"生成PPT大纲"
6. 可以下载生成的内容

## 📚 支持的教学方法

- **翻转课堂**: 学生课前自学，课堂进行讨论、实践和答疑
- **项目式学习**: 以真实项目为驱动，学生通过完成项目掌握知识
- **探究式学习**: 引导学生提出问题、探索和发现知识
- **合作学习**: 学生分组协作，共同完成学习任务
- **案例教学**: 通过分析真实案例，培养学生解决问题的能力
- **游戏化学习**: 将游戏元素融入教学，提高学习兴趣和参与度

## 🔑 获取 API Key

访问 [DeepSeek 开放平台](https://platform.deepseek.com/) 注册并获取 API Key。

## 📝 输出内容

### 教学设计包含：
- 教学目标（知识、能力、情感）
- 教学重难点
- 详细教学环节（时间、教师活动、学生活动）
- 教学评价方式

### PPT大纲包含：
- 每页标题和内容要点
- 建议的视觉元素
- 完整的演示逻辑
